## 🆘 Quick Troubleshooting

**Can't scan QR code?**
→ Make sure phone and computer are on same WiFi
→ Script uses tunnel mode which works across networks

**Script won't run?**
→ Mac/Linux: Run `chmod +x quick-review.sh` first
→ Windows: Just double-click `quick-review.bat`

**Need web version instead?**
```bash
npm run web
# Opens at http://localhost:19006
```
